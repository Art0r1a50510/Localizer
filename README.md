# Состав:
- localizertextpot.py — рекурсивно сканирует исходные файлы (.cpp, .h, .hpp, .cxx, .hxx, .cc, .hh), находит строки внутри макросов перевода и создаёт файл .pot (шаблон).

- librepottext.py — читает .pot файл, отправляет каждую строку на перевод в локальный сервер LibreTranslate и записывает результат в .po файл.
# Требования:
python 3.6 или выше
Библиотека requests (устанавливается через pip)
Сервер LibreTranslate (должен быть запущен локально или доступен по сети)

# Как использовать
1. Установка зависимостей
   ```bash 
   pip install requests
   ```

2. Запуск сервера LibreTranslate
Скачайте и установите LibreTranslate согласно [официальной инструкции](https://github.com/LibreTranslate/LibreTranslate#installation). Затем запустите сервер. 
По умолчанию сервер будет доступен по адресу `http://localhost:5000`.
3. Извлечение строк из исходников (`localizertextpot.py`)
   **Синтаксис:**
```bash
python localizertextpot.py <путь_к_исходникам> [выходной_файл.pot]
```

- `<путь_к_исходникам>` — корневая папка проекта (обязательно).
    
- `[выходной_файл.pot]` — имя создаваемого `.pot` файла (по умолчанию `messages.pot`).

**Пример:**
```bash
python localizertextpot.py D:\MyProject\src myproject.pot
```
После выполнения появится файл `myproject.pot` со списком уникальных строк.

4. Перевод `.pot` файла (`librepottext.py`)
**Синтаксис:**
```bash
python librepottext.py <входной.pot> [опции]
```
**Основные опции:**

| Параметр            | Описание                                                                      |
| ------------------- | ----------------------------------------------------------------------------- |
| `-o, --output FILE` | Имя выходного `.po` файла (по умолчанию `<язык>.po` в папке исходного `.pot`) |
| `-t, --target LANG` | Код целевого языка, например `ru`, `de`, `fr` (по умолчанию `ru`)             |
| `-s, --source LANG` | Код исходного языка (по умолчанию `en`)                                       |
| `-u, --url URL`     | Адрес сервера LibreTranslate (по умолчанию `http://localhost:5000`)           |
| `-d, --delay SEC`   | Задержка между запросами в секундах (по умолчанию `0.5`)                      |
**Пример:**
```bash
python librepottext.py myproject.pot -t ru -s en -u http://localhost:5000 -o ru.po
```
Скрипт переведёт все строки с английского на русский и сохранит результат в `ru.po`.
